**ipconfig**

* provides the local IP configuration
* **/all**
	* provides a more throughout overview of system information
* **/release (or /renew)**
* **/flushdns**
	* flushes out info regarding IP addresses
* **registerdns**

**netstat**

* shows us info about computers on the network
*  **-an | more**
	* shows us what ports our computer is listening on
	* the **more** command effectively paginates the response

**nslookup \[foobar.com\]**

* looks up nameservers

**arp**

*     provides a list of IP address each interface has communicated recently
* the column for "physical address" is the burned-in MAC address of an interface's card

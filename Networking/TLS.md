- Transport Layer Security
- encrypts
	- communication between web applications and servers
	- VOIP
	- email and other messaging
- Authenticates
- Verifies integrity of data

- first proposed by Internet Engineering Task Force (IETF)
- most recent version is TLS 1.3


- HTTPS implements TLS on top of plain HTTP

- TLS Certificate
	- a site or application needs a TLS certificate installed on the origin server
		- also called an SSL certificate
		- a cert authority issues the  cert to the person/business who owns a domain
		- cert contains info about how to validate the server's identity, such as who owns the domain and the server's public key
		- ***origin server*** -

**Firmware**
- BIOS - basic input output system - is the older system of managing computer firmware
- UEFI - Unified Extensible Firmware Interface is the current standard
	- supports 64 bit CPU operation
	- has graphical support in the menu 
	- can boot into legacy BIOS
	- common ways to access include hitting Exc, Delete, F1, F2, F10, or F12
	- you can shift click the Restart button from the windows Login screen to access the UEFI boot options

**boot options**
- network/PXE (Preboot Execution Environment) - uses the network adapter to retrieve boot instructions from a network server

* the standard issue database is 'postgres', and to access it type **psql -U posgres** and then the password you created when you first installed the Postgres software

* to create a new database, first follow the above to enter the posgres command line
	* type **CREATE DATABASE \[whaterver database name you want\]**

* to get general information on a given database and user type **\\conninfo** once you are logged into the Postgres command line

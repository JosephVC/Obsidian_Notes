- a mode of active directory that provides directory services for applications
	- provides a directory storage and the services for accessing the data store
		- What is directory storage?
			- the organized system of files and folders on a windows computer - comparable to an office filing system
	- uses standard APIs for accessing application data
		- these would be Active Directory, active directory services interfaces, and others
		- operates independntl of active directory and AD domains and forests
	- works as either a standalone data store or oprates with replication
	- concentrates on the requirement of specific application rather than Windows as a whole

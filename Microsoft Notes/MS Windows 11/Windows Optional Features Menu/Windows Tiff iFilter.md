- Windows TIFF iFilter is a tool that enables Optical Character Recognition (OCR) on TIFF files, allowing for indexing and full-text search of these documents. It is available in all versions of Windows 11, including the Home edition.

- Tiff: Tag Image File Format
	- used to store raster graphics + image information
	- lossless form of file compression, so the file size is large but there is no drop in quality
	- stores additional layers in an image
	- ### Advantages of TIFF files.

		- Predominantly lossless compression means TIFF files retain the original image’s detail and color depth — perfect for high-quality professional photos.
		
		- Their impressive detail means TIFFs are ideal for high-resolution scans, such as backing up your handmade artwork and personal documents.
		
		- TIFFs are universal and adaptable as a file format, so can be used with the major operating systems.
		
		- The file can work as a container for smaller-sized JPEGs, storing multiple images in one master raster graphic.
		
		- TIFFs are a good option to store high-resolution images ahead of editing and asset creation.
		
		- The Tiff file format can accommodate from 1 bit (black and white to 32 bits, with an Alpha channel  
		    Making it either 8 bits per channel + an 8 bit transparency Alpha channel for RGBA 32 bits, or 16 bits per channel for RGB. Perfect for HDR photos.
		
		- The size limit for a tiff file is 4GB which makes it an excellent alternative from proprietary file format to hold a bigger amounts of data
		
		- The Tiff file format can accept different colour models such as CMYK, RGB, CIE lab etc.
		
		- It also retain spot colours used in the printing industry for printing special inks or finishes.
		
		- **Other accepted colour models are:**
		
		**Grayscale: For images that only contain shades of gray.**
		
		**Indexed Color: Uses a color palette to represent images with a limited number of colors.**
		
		**YCbCr: Commonly used in video and digital photography, representing color as brightness (Y) and two color difference components (Cb and Cr).**
		
		- It can accept transparency the same way as a PNG-24 does.

### Disadvantages of TIFF files.

- Detail and resolution lead to TIFFs being quite large files. They can take up valuable space on your drive.

- Their size makes them difficult to share or send to contacts or clients.

- Their high quality makes them a poor choice for website design, since detailed images can slow a website’s loading speed. Lighter file formats, like a [JPEG](https://www.adobe.com/creativecloud/file-types/image/raster/jpeg-file.html), might be more useful for online images.

- Certain features can be read differently by different software, notably adjacent ones like text, vector paths, and non essential tags.

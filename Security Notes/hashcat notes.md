**Get hashes on a single line**

For hashcat to work best, you should have the hash on a single line - the following site was useful for doing this consistently:  <https://www.textfixer.com/tools/paragraph-to-lines.php>

**The password is at the end of the hash**

hashcat will  put the cracked password at the end of whatever hash you fed it:

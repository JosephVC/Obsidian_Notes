* **ARP cache poisoning** involves convincing a target's machine that we have become its **gateway**, and we also convince the gateway that in order to reach the target machine all traffic must go through us.  ARP = Address Resolution Protocol
	
* every computer on a network maintains an ARP cache that stores the most recent MAC (Media Access Control) addresses matching the IP addresses on the local network.  this cache is poisoned with entries we control

domains are leased out by companies called **Domain Registrars** 

the **whois** command allows you to see who a domain is **registered to** 

the command is run via **whois \[some domain\]**

When using **[whois.com](http://whois.com)** and getting results, keep in mind that some credentials my be obscured to prevent scraping/copying:

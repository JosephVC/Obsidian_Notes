# Arrays

arrays are marked with **\[ \]**
an array of numbers is defined by **numbers\[\]**
arrays can contain any type

## note on how booleans are handled in arrays

![[./_resources/Untitled_Note.9.resources/typescript - arrays - booleans.jpg]]
while there are arrays that handle multiple types, for now we deal with arrays that handle only one type
**note that you need a colon after your variable to define the array, then the type of array (strings, numbers, booleans, etc.), then an equals to what the array contains**
![[./_resources/Untitled_Note.9.resources/typescript - arrays - type checking.jpg]]
apparently, giving your array the designation of **string\[\]** or **number\[\]** defines the type that your array can accept
![[./_resources/Untitled_Note.9.resources/typescript - arrays - single type allowed.jpg]]

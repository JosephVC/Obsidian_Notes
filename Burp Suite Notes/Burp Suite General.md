**What is Burp**

* a proxy to send web traffic through
* requests can be sent and resent, change requests, and intercept requests before they're sent

**Why use Burp?**

* many people have a need for web-based pentesting
	* the ability to edit requests sent is key to testing an application
	* Burp provides a whole IDE and range of features to do this
		* Repeater = repeats request
		* Intercept = intercepts a request before it happens
		* Intruder = brute force tools
